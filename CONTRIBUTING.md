# 🤝 অবদান রাখার নিয়মাবলী (Contributing Guide)

CryptoFlip-এ অবদান রাখতে চাওয়ার জন্য ধন্যবাদ! নিচের নিয়মগুলো অনুসরণ করুন।

## শুরু করার আগে

1. এই রিপো fork করুন
2. আপনার মেশিনে clone করুন:
   ```bash
   git clone https://github.com/your-username/cryptoflip.git
   cd cryptoflip
   ```
3. `README.md`-এ দেওয়া ইন্সটলেশন গাইড অনুসরণ করে লোকাল এনভায়রনমেন্ট সেটআপ করুন

## ব্রাঞ্চ তৈরি করা

নতুন ফিচার বা বাগফিক্সের জন্য সবসময় নতুন ব্রাঞ্চ তৈরি করুন:

```bash
git checkout -b feature/your-feature-name
# অথবা
git checkout -b fix/bug-description
```

## কোড স্টাইল

- **TypeScript** ব্যবহার করুন, `any` টাইপ এড়িয়ে চলুন
- ফাংশন ও জটিল লজিকে বাংলা কমেন্ট যোগ করুন (এই প্রজেক্টের ঐতিহ্য অনুসরণ করে)
- কমিট করার আগে `npm run lint` চালান

## কমিট বার্তা

স্পষ্ট ও বর্ণনামূলক কমিট বার্তা লিখুন:

```
feat: Squad Flip-এ ৬ষ্ঠ সদস্য যোগের সাপোর্ট
fix: Crypto Rain ক্লেইম বাটনে ডাবল-ক্লিক বাগ ঠিক করা হয়েছে
docs: README-তে Docker সেটআপ নির্দেশনা আপডেট
```

## Pull Request পাঠানো

1. আপনার পরিবর্তন push করুন
2. GitHub-এ Pull Request তৈরি করুন
3. PR বর্ণনায় কী পরিবর্তন করেছেন ও কেন তা ব্যাখ্যা করুন
4. সম্পর্কিত ইস্যু থাকলে লিংক করুন (`Closes #123`)

## নিরাপত্তা সংক্রান্ত সমস্যা

Provably Fair সিস্টেম, পেমেন্ট লজিক, বা অথেন্টিকেশনে কোনো নিরাপত্তা দুর্বলতা পেলে **পাবলিক ইস্যু খুলবেন না**। সরাসরি মেইনটেইনারদের সাথে যোগাযোগ করুন।

## প্রশ্ন আছে?

একটি Issue খুলে জিজ্ঞাসা করুন — আমরা সাহায্য করতে পেরে খুশি হব!
